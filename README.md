# KillProcess

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kill_process'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kill_process

## Usage

```ruby
KillProcess.after(10)
```
